Head Section Tags

The link Tag

The <link> tag allows assets defined in external files to be linked to an HTML document.

<!-- Add a favicon -->
<link rel="icon" href="itvclock.ico">

<!-- Add a stylesheet -->
<link rel="stylesheet" type="text/css" href="itvclock.css">

<!-- Add a link to an author page -->
<link rel="author" href="https://plus.google.com/u/0/+MarkTwain/about/p/pub" title="Mark Twain">

The script Tag

The <script> tag is used to add JavaScript files.

<!-- HTML 4 -->
<script src="itvclock.js" type="text/javascript" />

<!-- HTML 5 -->
<script src="itclock.js">

To process the JS after rendering the document, use the 'defer' parameter.

<!-- HTML 4 -->
<script src="itvclock.js" type="text/javascript" defer="defer" />

<!-- HTML 5 -->
<script src="itvclock.js" defer="defer">

To process the JS and render the HTML at the same time, use the 'async' parameter.

<!-- HTML 4 -->
<script src="itvclock.js" type="text/javascript" async="async" />

<!-- HTML 5 -->
<script src="itvclock.js" async="async">

The noscript Tag

The <noscript> tag can be added alongside the script tag.

<noscript>No JavaScript Support; Please Enable JavaScript!</noscript>

Embedding JS Code

To embed JS code directly in the page:

<head>
    <script>
        Document.getElementById("JSapp"). innerHTML="Hello World JavaScript";
    </script>
</head>

The style Tag

The <style> tag is used to add inline styles.

<head>
    <style type=text/css>
        h1 { color: green; }
    </style>
</head>
